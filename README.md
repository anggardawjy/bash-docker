# Bash Scripting Docker Install
Script Automation Install Docker CE

Recommended : 
---------------
- Ubuntu 22.04
- Debian 10
- Centos 9 Stream
- Rocky Linux 9
- Almalinux

Configuration :
---------------
- sudo apt install git -y
- git clone https://github.com/anggardawjy/bash-docker.git
- cd bash-docker
- chmod -R 777 *
